Unity Systems


Random 3D Dungeon Generation

Dungeons generator.
Using multiple Tile map objects allows for level generations with different models and zones.
Dungeons are allowed to grow dynamically at runtime.
Dungeons can also be generated vertically.
Pre-created models, which are used to generate the map based on a predefined number of connections.
❮ ❯

Challenge

  • Keeping track of an unstable number of connections, which change through level generation.
  • Allow for room models to be changed, increased or decreased in number.

Solutions

  • Used recutiuon to continuously loop though all open connections. Doing so reduces memory use, as no data is stored during generation. As well as decreasing generation time.
  • Designers can place connection points in each room. These points are used to determine where room connections need to be made.
  • Stored all information into a grid to prevent room intersections and allow for saving of level layout. The possess is done after level generation to prevent performance loss during the generation process.
